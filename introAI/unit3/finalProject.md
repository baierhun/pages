# 🟢 **Final Project: Build Your Own Website**

## 🌐 **Project Overview**

You will create a **brand new website** using:

* **HTML** (structure)
* **CSS** (design)
* **JavaScript** (interactivity)

Your final product should be a **complete, polished website** that works and looks consistent.

---

## 🎯 **Goal**

Build a website that:

* Is organized and functional
* Looks visually consistent
* Includes interactive features

---

## 📦 **Repository Requirement**

This project must be created in a **new repository**.

You must:

* Create a **new repo for this project** (do not reuse an old one)
* Build your new website **from scratch**
* Keep all project files organized inside this repo

---

## 🎨 **Required: Consistent Theme**

Your website must have a **clear, consistent design across all pages**.

### This includes:

* A consistent **color scheme**
* Consistent **fonts and text styles**
* A similar layout/style on each page

Your site should feel like **one complete product**, not separate pieces.

---

## 📋 **Project Requirements**

### 🧱 **HTML (Structure)**

Your site must include:

* At least **3 pages**
* Proper HTML structure
* The following elements:

  * Headings
  * Paragraphs
  * Links
  * Images
  * A list (ordered or unordered)

---

### 🧭 **Navigation Menu (Required)**

Your website must include a **menu bar on every page** that:

* Appears in the same place on each page
* Links to **all pages in your site**
* Looks consistent across the entire website

---

### 🎨 **CSS (Styling)**

You must use CSS to style your website. Include:

* A background color or image
* Text styling (font and color)
* Spacing (margin or padding)
* **Consistent styling across all pages**

---

### ⚡ **JavaScript (Interactivity)**

Include at least **2 interactive features**, such as:

* A button that changes text or content
* A color/theme change
* Show/hide content
* A pop-up message (alert)
* A response to user input

---

## 🤖 **Using AI (Vibe Coding)**

You may use AI to help, but you should:

* Ask for **specific features or improvements**
* Make sure you understand what is being added
* Use AI to improve your **design and functionality**

---

## 🪜 **Suggested Workflow**

1. Create your new repository
2. Plan your pages and theme
3. Build your HTML pages
4. Add your menu bar to every page
5. Apply consistent CSS styling
6. Add JavaScript features
7. Test your website

---

Submit:

* Open correctly
