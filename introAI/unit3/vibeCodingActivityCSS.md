# 🟢 Vibe Coding – Part III

## 🎨 Assignment: Design Your Website with CSS

---

## 🎯 Your Goal

Use AI to design the **look and feel (vibe)** of your website.

You will:

* Describe what you want
* Use AI to generate CSS
* Store your CSS in a **separate file**

---

## 📂 What You Need to Create

You will have:

* `index.html` (your website)
* `styles.css` (your design file)

---

## ✅ Step 1: Pick a Vibe

Choose one:

* Dark mode
* Bright / colorful
* Clean / minimal
* Gaming
* School / academic
* Professional

---

## ✅ Step 2: Describe Your Vibe

Think about your own description (no AI yet):

> “I want my website to look ______________________________
> ______________________________________________________”

---

## ✅ Step 3: Build Your Prompt

Use this structure:

```
Here is my HTML. Create CSS for a separate styles.css file.

I want:
- Colors:
- Fonts:
- Spacing:
- Layout:
- Images:
```

Fill in each part with your ideas.

---

## ✅ Step 4: Use AI

* Paste your HTML
* Paste your prompt
* Generate CSS

---

## ✅ Step 5: Create Your CSS File

1. Create a new file called:

```
styles.css
```

2. Copy the CSS from AI and paste it into that file

---

## ✅ Step 6: Connect CSS to HTML

Inside your `<head>` section, add:

```html
<link rel="stylesheet" href="styles.css">
```

---

## ✅ Step 7: Test Your Design

* Save both files
* Refresh your website

👉 If nothing changes, check:

* File name is correct (`styles.css`)
* The `<link>` is inside `<head>`

---

## ✅ Step 8: Improve Your Design

Ask AI to improve specific things:

* “Fix the spacing”
* “Change the font”
* “Make colors less bright”
* “Center the layout”

⚠️ Do NOT say: “make it better”
Be specific.

---
