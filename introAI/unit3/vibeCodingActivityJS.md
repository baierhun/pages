# 🟣 Vibe Coding Part IV Assignment

## 💻 Make Your Website Interactive (JavaScript)

---

## 🎯 Objective

You will use **JavaScript + AI** to make your website **interactive**.

By the end, your site should **respond to user actions** (clicks, input, etc.).

---

## ⚠️ Rules

* You are allowed to use AI (ChatGPT, Gemini, etc.)
* You must **understand what your code does**
* You may NOT just copy/paste without testing
* Work **step by step**

---

## 🚀 Step 1: Set Up JavaScript

### 📝 Task

Add a `<script>` section to your HTML file.

👉 Ask AI:

> “How do I add JavaScript to my HTML file?”

### ✅ Goal

Your page should now be able to run JavaScript.

---

## 👆 Step 2: Button + Click Event

### 📝 Task

Create a button that shows a message when clicked.

👉 Ask AI:

> “Make a button that shows an alert when clicked”

### ✅ Goal

* A button appears
* When clicked → a message pops up

---

## 🔄 Step 3: Change Something on the Page

### 📝 Task

Make a button that changes something on your page.

Examples:

* Change text
* Change color
* Show/hide something

👉 Ask AI:

> “Make a button that changes the text on my page”

### ✅ Goal

Something on your page **updates without refreshing**

---

## 📥 Step 4: User Input

### 📝 Task

Let the user type something and use it.

Examples:

* Enter their name
* Display a message using their input

👉 Ask AI:

> “Take a user’s name from an input box and display a welcome message”

### ✅ Goal

* User types something
* Website responds using that input

---

## 🧠 Step 5: Add Logic (if / else)

### 📝 Task

Make your site make a decision.

Examples:

* Different message based on input
* Simple yes/no logic

👉 Ask AI:

> “Use if else to show different messages depending on what the user enters”

### ✅ Goal

Your site shows **different results depending on conditions**

---

## 🎲 Step 6: Add Something Fun

### 📝 Task

Add ONE of the following:

* Random fact generator
* Random background color
* Simple quiz question
* “Magic 8-ball” style answer

👉 Ask AI:

> “Make a random [idea] generator using JavaScript”

### ✅ Goal

Your site does something **unexpected or dynamic**

---

## 🧪 Step 7: Test Everything

### 📝 Task

* Click every button
* Try different inputs
* Fix anything broken

👉 Ask AI:

> “Check my JavaScript and suggest improvements”

### ✅ Goal

Your site works without errors

---

## 🎨 Step 8: Make It Your Own

### 📝 Task

Customize your site:

* Add more buttons
* Add better messages
* Combine features

👉 Ask AI:

> “How can I improve my interactive website?”

---

## 📦 Final Requirements

Your website must include:

✅ At least **1 button with a click event**
✅ At least **1 change to the page (text, color, etc.)**
✅ At least **1 user input feature**
✅ At least **1 if/else decision**
✅ At least **1 fun/dynamic feature**

---

## 💬 Reflection (Short Answer)

Answer in 1–2 sentences each:

1. What did you make your website do?
2. What was the most interesting thing JavaScript allowed you to do?
3. What is one prompt that worked really well for you?

---

## 🧠 Reminder

> You are not learning JavaScript by memorizing.
> You are learning how to **build with it using AI**.
