# 🟡 Vibe Coding – Part II

## Understand + Expand Your Website

In Part I, you built your first website.

In **Part II**, your goal is to:

* Understand the code you already wrote
* Practice explaining HTML
* Add more content using new HTML tags

> ⚠️ You may NOT use AI for the commenting steps in this assignment.

---

## Step 1. Comment Your Entire Website

Open your `index.html` file.

Add a comment to **every line of code** explaining what it does.
> ⚠️ Do NOT comment the lines in the `<style>` or `<meta>` tags.

### Example:

```html
<h1>My Website</h1> <!-- This is the main heading -->
<p>Welcome to my site!</p> <!-- This is a paragraph -->
<img src="mypic.jpg" alt="My photo"> <!-- This is an image -->
<a href="https://example.com">Visit Example</a> <!-- This is a link -->
```

### Requirements:

* Every line must have a comment
* Comments must be in your own words
* Do NOT use AI for this step

---

## Step 2. Add a New Section

Add at least **one new section** to your page.

### You must use:

* `<h2>` for the title
* `<p>` for text

### Ideas:

* About Me
* My Favorite Things
* Top 3 Games
* My Hobbies

---

## Step 3. Create a List (New Skill)

Add a list with at least **3 items**.

### You can choose:

**Unordered list (bullets)**

**OR**

**Ordered list (numbers)**

---

## Step 4. Add More Images

Add at least **2 new images** to your website.

### Requirements:

* Each image must include `alt` text
* Place images in a relevant section

### Example:

```html
<img src="cat.jpg" alt="A cute cat">
```

---

## Step 5. Add More Links

Add at least **2 new links** to your site.

### Requirements:

* Use meaningful link text (NOT “click here”)
* Links should go to real websites

---

## Step 6. Comment Your New Code

Go back and add comments to **everything you added**.

* Lists
* Images
* Links
* Sections

> Every new line should be explained in your own words.

---

## ✅ Final Checklist

Before submitting, make sure you have:

* ✅ Commented every line of your original code
* ✅ Added at least 1 new section
* ✅ Added a list (3+ items)
* ✅ Added 2 new images (with alt text)
* ✅ Added 2 new links
* ✅ Commented all new code
* ✅ Everything is working

---

## 🚫 Important Rules

* No AI for commenting
* You may use AI for debugging **after** commenting is complete
* All work must be your own

---

## 🎯 Goal

By the end of Part II, you should be able to:

* Read your HTML and explain what it does
* Add new content without copying blindly
