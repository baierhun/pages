# Making a Webpage with AI

## 🚀 Getting Started with Vibe Coding

### Before we Begin
Remember, AI is not perfect. It will **not** give you perfect information every time. If something is unclear, explain to it that you are seeing something different and continue to use it to work through problems to fully complete the task. 

We will use Google's Gemini as the AI for this assignment.

* Go to [gemini.google.com](https://gemini.google.com)
* Make sure to sign in with your Google account
* At the prompt, in the bottom right you will see a dropdown that may say `Fast` or `Thinking`. Make sure `Thinking` is selected.

### Step 1. Create Github Account
Paste this into the AI:

> I am brand new to coding and GitHub. I want to build a website using GitHub Pages.  
> Give me step-by-step instructions to:
> 1. Create a GitHub account  
> 2. Sign in using the "Sign in with Google" option (include exactly where to click and what to expect)  
>  
> Assume I have never done this before and include clear, beginner-friendly steps.

---

### Step 2. Create Repository
Paste this into the AI:

> Now help me create my first GitHub repository for a website.  
> Give me step-by-step instructions to:
> 1. Create a new repository  
> 2. Name it "demo-page"
> 3. Make it public  
> 4. Initialize it properly  
>  
> Explain what each step does in simple terms.

### Step 3. Setup Github Pages
Paste this into the AI:

> Help me turn my repository into a live website using GitHub Pages.  
> Give me step-by-step instructions to:
> 1. Enable GitHub Pages  
> 2. Choose the correct branch and folder  
> 3. Find my live website URL, bookmark it, and leave the tab open for later! 
>  
> Explain what GitHub Pages is in one simple paragraph.

---

### Step 4. Hello, Webpage!
Paste this into the AI:

> Help me create my first webpage.  
> Generate a simple HTML file called `index.html` that includes:
> - A title for my page  
> - A heading with our class name  
> - A short paragraph about the class 
>  
> We will be doing everything through the web interface. Explain exactly how to create this file in my GitHub repository.

---

### Step 5. Edit My Page
Paste this into the AI:

> My website is live, but I want to update it.  
> Show me step-by-step how to:
> 1. Edit my `index.html` file in GitHub  
> 2. Commit the changes  
> 3. Refresh my site to see updates  
>  
> Explain what a "commit" means in simple terms.

---

### Step 6. Style My Webpage
Paste this into the AI:

> Help me improve my website design.  
> Add simple styling (colors, fonts, spacing) using CSS directly in my HTML file.  
> Keep it beginner-friendly and explain what each part of the code does.

### Step 7. Add an Image
Paste this into the AI:

> Help me add an image to my website.  
> Generate HTML code to:
> - Insert an image onto my page  
> - Include alt text  
> - Resize the image so it fits nicely  
>  
> Then explain:
> 1. Where I should get an image from (free sources)  
> 2. How to upload the image file to my GitHub repository  
> 3. How to make sure the file path works correctly  

---

### Step 8. Add a Link
Paste this into the AI:

> Help me add a clickable link to another website.  
> Generate HTML code that:
> - Creates a link to an external website (like a favorite site of mine)  
> - Makes the link open in a new tab  
>  
> Then explain how links work in HTML in simple terms.

---

### Step 9. Clean Up
Paste this into the AI:

> I built a simple website using HTML. Here is my current code:
>  
> (PASTE YOUR FULL index.html CODE HERE)
>  
> Please review my code and:
> 1. Check for any errors or things that might break  
> 2. Suggest improvements for readability and organization  
> 3. Suggest small design improvements (colors, spacing, layout)  
> 4. Make sure my image and link are working correctly  
>  
> Keep your feedback beginner-friendly and explain why each suggestion helps.
