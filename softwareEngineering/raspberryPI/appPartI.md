# **Java Lanterna GUI + Raspberry Pi API Assignment**

## **Overview**

You will build a Java terminal-based GUI using Lanterna that connects to your Raspberry Pi REST server.

You will use a **template GitHub repository** to get started.

Your app should run locally and make requests to your Raspberry Pi endpoints.

---

## **Part I – Setup + First Request**

Clone <a href="https://github.com/baierhun/apiApp" target="_blank" rel="noopener noreferrer">this template repo</a> (click on the green `Use this template` button in the top right corner) and get the application running locally on your computer.

Modify the app to make a request to your Raspberry Pi server and display the response.

### **Requirements**

* Template project runs successfully
* App makes a request to your Raspberry Pi
* Response is displayed in the GUI

### **Checkpoint**

* You must **demo your working app** to me before moving on

---

## **Part II – Interactive GUI**

Expand your app to allow user interaction.

### **Requirements**

* GUI includes at least one interactive element (button, menu, or input)
* User action triggers a request to your Raspberry Pi
* GUI updates based on the response
* App connects to at least two different endpoints

### **Checkpoint**

* You must **demo your working app** to the teacher

---

## **Guidelines**

* Use the provided **template repository**
* You are responsible for learning:

  * How Lanterna works
  * How to make HTTP requests in Java
* You may adapt examples
* You may work with a partner

---

## **Challenge (Optional)**

* Display formatted JSON data in the GUI
* Add multiple screens or views
* Handle errors (server down, bad response, etc.)
* Allow user input to be sent to the Raspberry Pi
