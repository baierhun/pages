# **Raspberry Pi GPIO Assignment**

## **Overview**

You will use GPIO pins on a Raspberry Pi to read input from a button.

You are expected to find your own schematics and code online.

---

## **Part II – Button-Controlled LED**

Create a circuit and program where an LED turns on when a button is pressed.

### **Requirements**

* Uses a push button as input
* LED turns on when pressed
* Uses GPIO input and output

### **Checkpoint**

* You must **demo your working circuit** to me

---

## **Guidelines**

* Find your own:

  * Wiring diagrams
  * Code
* You may adapt examples

---

## **Challenge (Optional)**

* Toggle LED on/off with each button press
* Add another LED
* Organize your code with functions
