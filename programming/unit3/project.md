## Repetition-Based Program

**(Using Loops)**

### Skills You Must Demonstrate

* `for` or `while` loop
* Counter variable
* Accumulator variable
* Repeating actions

---

## Project Description

Create a program that repeats actions using a loop.

The program must collect multiple pieces of information or repeat an activity until a condition is met.

---

## You May Choose Any Theme

Examples:

* Survey collector
* Shopping cart calculator
* Number guessing game
* Workout tracker
* Attendance counter
* Class average calculator
* Multi-round game

---

## Requirements

Your program must:

* Use at least **one loop**
* Use at least **one accumulator variable**
* Use at least **one counter variable**
* Display a final summary at the end
* Run more than once (not hard-coded)
