## Decision-Based Program

**(Using Conditionals)**

### Skills You Must Demonstrate

* `if`
* `elif`
* `else`
* Comparison operators (`>`, `<`, `==`, etc.)
* Logical operators (`and`, `or`)

---

## Project Description

Create a program that makes decisions based on user input.

The output must change depending on what the user enters.

---

## You May Choose Any Theme

Examples:

* Personality quiz
* Grade calculator
* Ticket pricing system
* Choose-your-own-adventure story
* Loan approval simulator
* Restaurant recommendation system
* Game outcome predictor

---

## Requirements

Your program must:

* Include at least **3 decision branches**
* Use at least **one `and` or `or` statement**
* Clearly explain results to the user
* Have meaningful differences between outcomes
