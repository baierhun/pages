# 🧩 Final Project: Escape!

## 🎯 Objective

Create a **text-based escape room game in Python** where the player must solve puzzles to escape.

You will use everything we’ve learned this course: variables, input/output, conditionals, loops, and functions.

---

## 📋 Requirements

### 🔹 Game Design

Your game must include:

* A **beginning** (introduction/story)
* At least **3 puzzles or clues**
* At least **2 mini-games**
* A **final outcome** (escape or fail)

---

### 🔹 Programming Requirements

Your code must include:

* ✅ Variables (`int`, `string`, `boolean`)
* ✅ `input()` and `print()`
* ✅ At least **3 conditionals** (`if`, `elif`, `else`)
* ✅ At least **1 while loop**
* ✅ At least **2 functions**
* ✅ At least **1 use of logical operators** (`and`, `or`, `not`)
* ✅ Replay options for all games


Your code must not include

* ❌ Concepts we have not covered in class
    * This includes, but is not limited to, `for` loops, `break` statements, `range`, lists/collections of any kind (dictionaries, tuples, lists, etc...)
* ❌ **Any** AI assistance whatsoever
    * ❌ You may **not** ask AI for help in any way on this project
    * ✅ If you need help, come see me during extra help.
* ❌ Code written by other people
    * This should be entirely your own work

*The consequences for not following these instructions could result in a 0 and/or disciplinary action from the Dean.*

---

### 🔹 User Experience

* Instructions must be **clear**
* The user must be able to **retry** incorrect answers

---

## 🧩 Puzzle Ideas (Choose Your Own!)

You can use ideas like:

* 🔐 **Code Lock** (guess a number/code)
* 🧠 **Riddle**
* 🎲 **Guessing Game**
* ➕ **Math Challenge**
* 🔑 **Choice Puzzle** (pick the correct item)
